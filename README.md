# History2Data

基于[中國歷代人物傳記資料庫](https://projects.iq.harvard.edu/chinesecbdb)的个人兴趣研究，希望能用程序化的方式，展现这些历史人物之间的关系

## 程序相关
> 具体的参数详见配置文件

> 数据库请自行从中國歷代人物傳記資料庫(CBDB)官网下载

### 语言
+ Java

### 类库和框架
+ SpringBoot
+ JooQ

### 数据库
+ SQLite
